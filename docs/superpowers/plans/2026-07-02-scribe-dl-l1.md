**Goal:** `listings/ch561_scribe_dl.rune`: the scribe display list as an
inductive type, a total pure parser `parse : String -> Result (List Op) PErr`
for the ps postfix language with line:col diagnostics, a printer
`print : List Op -> String`, refl round-trip proofs, plus harness, emit, and
REPL acceptance tests.

**Constraints (from repo conventions):** self-contained listing (redeclare
preamble verbatim: Nat + builtin nat/natAdd/natMul accel, Bool, Unit, Bytes/
String/codeOf, Option, List, Result); constructor declaration order
load-bearing (nil before cons, false before true); no general recursion in
total defs (eliminators + fuel-bounded `case ... with ih`); `foreign` ops are
neutral in the checker so the parser/printer must be PURE rune (no byteLen
host op) for refl proofs to normalize; strings are packed bignums with 0x01
sentinel.

## Design pinned up front

- **Numbers**: exact decimal record `data Num : U is num : Bool -> Nat ->
  Nat -> Nat -> Num end` (neg, units, fracLen, fracDigits), canonical form:
  no trailing zero in frac (fracDigits % 10 != 0 unless fracLen 0), no
  negative zero. Denotes the rational (units*10^fracLen + fracDigits) /
  10^fracLen, negated when neg. Covers every literal the Go printer emits for
  the L1 corpus; exponent syntax is out of scope and yields a diagnostic.
- **Color**: `data Color : U is rgba : Nat -> Nat -> Nat -> Nat -> Color end`.
- **Enums**: CornerStyle (circular|continuous), CapT (buttC|roundC|squareC),
  JoinT (bevelJ|roundJ).
- **Op**: 20 constructors mirroring Go dl exactly (opMoveTo .. opClear).
- **Pure string kit** (ch439/ch213 idiom): strNull/strHead/strTail/strCons +
  pure strLen (fuel) + `strCat a b = bytes (codeOf a + 256^strLen(a) *
  (codeOf b - 1))` with pure pow. No foreign ops anywhere in the codec.
- **Tokenizer**: fuel loop over bytes tracking line:col; skips spaces, tabs,
  CR, LF (line++), `%` comments; emits `mkTok line col word` with word
  packed via reverse-accumulate + finalize.
- **Parser**: classify token (digit/'-' -> number, '#' -> color, else name);
  stack machine `List Val` (vNum|vCol|vName) folded left over tokens via the
  function-accumulator ListElim pattern; Result short-circuit; gsave counter;
  end check: empty stack. `data PErr is mkPErr : Nat -> Nat -> Nat -> PErr`
  (line, col, errCode) with named errCode constants (errUnknownWord,
  errMissingArg, errWrongType, errBadColor, errBadNumber, errGrestore,
  errLeftover, errBadStyle, errBadCap, errBadJoin).
- **Printer**: opLine per constructor; program joined with `\n` (trailing
  newline like Go ps.Print); numbers via fuel divmod-10 digits (ch527
  pattern), colors as #rrggbb / #rrggbbaa lowercase hex.
- **Round trip**: refl proofs `Eq (Result (List Op) PErr) (parse (print P))
  (ok ... P)` for concrete programs covering all 20 ops, both corner styles,
  all caps/joins, negative and fractional numbers, alpha colors, comments and
  CRLF in hand-written source.
